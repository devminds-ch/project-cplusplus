# C++ Training Project by [devminds GmbH](https://devminds.ch)

## Directory structure

```
├── build       Reserved folder for build artifacts
├── cmake       Custom CMake files
├── docs        Documentation folder
├── external    3rdparty libraries as Git submodules
├── src         Source code
├── tests       Tests
└── tools       Scripts
```
